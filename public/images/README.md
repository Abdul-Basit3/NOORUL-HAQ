# Images Directory

Place the following images in this directory:

## Required Images

### 1. mosque-background.jpg
- **Purpose**: Background image for admin login page
- **Recommended Size**: 1920x1080px or higher
- **Format**: JPG, PNG, or WebP
- **Description**: A beautiful mosque or Islamic architecture image

### 2. school-logo.png
- **Purpose**: School logo displayed on admin login and navbar
- **Recommended Size**: 512x512px (square)
- **Format**: PNG with transparent background
- **Description**: Your school's official logo

### 3. organization-logo.png
- **Purpose**: Organization/Mosque logo displayed on admin login and navbar
- **Recommended Size**: 512x512px (square)
- **Format**: PNG with transparent background
- **Description**: Your organization's official logo

## Optional Images

### 4. placeholder-video.jpg
- **Purpose**: Placeholder for lecture videos without thumbnails
- **Recommended Size**: 1280x720px (16:9 ratio)

### 5. placeholder-course.jpg
- **Purpose**: Placeholder for courses without thumbnails
- **Recommended Size**: 1280x720px (16:9 ratio)

### 6. placeholder-avatar.jpg
- **Purpose**: Placeholder for user profiles without photos
- **Recommended Size**: 400x400px (square)

## Image Optimization Tips

1. **Compress images** before uploading to reduce file size
2. Use **WebP format** for better compression and quality
3. Provide **multiple sizes** for responsive images
4. Use **lazy loading** for images below the fold
5. Add **alt text** for accessibility
