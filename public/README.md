## Image Specifications

### Recommended Dimensions:
- **Executive Photos**: 400x400px (square, will be displayed as circles)
- **Activity Images**: 800x600px (landscape)
- **Project Images**: 800x600px (landscape)
- **Course Images**: 600x400px (landscape)
- **Lecture Images**: 800x450px (16:9 ratio)
- **Logos**: 200x200px (square or transparent background)

### File Format:
- JPEG (.jpg) for photos
- PNG (.png) for logos with transparency (optional)

### File Size:
- Keep images under 500KB for optimal loading
- Compress images before uploading

## Fallback Behavior

If an image fails to load or is missing, the application will display:
- **Executive Photos**: Default avatar icon
- **Activity/Project/Course/Lecture Images**: Gradient background with icon
- **Logos**: Default icon

This ensures the application remains functional even if some images are missing.
